    丙午

    # 丙午

    # 沁園春·假日

    冷峻駁紋，竭能有限，元始之湯。話一言「宜有」，萬靈生滅，欲工神筆，恰構繁邦。酬室過勞，積疲成翳，碌碌轍塵爲計忙。便千世，無適逢所向，可不倉惶。

    卷軸似此綿長。委真理茫茫覓四方。莫故封自止，被殘體制，極聲窮利，豈熨創傷。但座環中，瓊樓影裏，笑語華胥大夢藏。付斯願，共弟兄我屬，置酒輿央。

    # 星象

    年終遣興望穹庭。一宇清濁問帝星。分野山川皆了了，微茫樞幄鮮錚錚。斬殺豈在長安殿，恐虐非關萬歲生。億兆赤心傾榜首，何須從景見龍徵。

    # 占卜官

    惶恐登壇究大衍，措祈設祭奉一人。叨叨咒詛欃槍衆，爾爾周期溫馴民。蓍草燭臺風動響，聖神父子殼開紋。喜得盛世兼吉兆，了解越多擁護深。

    # 聞軍樂戲書

    軍歌こそ知るから和歌はイージーだ

    # On Femininity

    昔者弗翁濟疑難，也將癔病比夷蠻。熒屏若共陰曹賞，孔孟猶尊塔利班。

    # 揚州慢·盜跖

    丙午年前，余讀《南華·雜篇·盜跖》，歎惟儒則迷使天下學子盡以入仕途、圖宦達、求富貴爲至大矣。儒生詐言虛語，媾合建制，則於我華夏，可謂貽誤千年也已。乃行櫽括爲詞。

    父必教兒，兄能詔弟，一時禮樂尊卑。弟雖天下害，丘請往而說。再謁諫、將軍颯爽，三德兼具，宜順帷幃。見將軍、案劍足展，勃怒瞋眉。

    跖談世事，笑恆民、愚陋皆非。數臣仕無德，帝王無道，終作塵灰。詐巧更迷天下，奚足論、儒本國賊。子遁逃觳觫，三失車轡而歸。

    # 聞伏誅

    「次は誰」獨夫らが問う地獄にて

    # 毛概課上

    聖哲坐鎮舜堯邦。實踐理論相益彰。民主焉能妥協到，聯合方有團結強。類之爭鬭千年在，活的靈魂萬丈長。羣衆作爲創世主，大幅邁入自由疆。

    # 傷風寒

    荅焉枯槁有由緣。稍怠東風暑遽連。速朽周君誤因晚，特抓林帥茸跟參。幾從病榻絕喪亂，一臥他鄉記疫年。勒忒遊人驚坐起，蒙誰祝聖滿青衫。

    # 贈友人戲書

    問君何爲理紛紜。大亂但求全一身。餘电足能完舊影，濁醪堪備久沈吟。素知此土頑攻匪，試看他時秦滅秦。捧逗員輿千萬場，誰非花落護荊榛。

    # 桀頌

    每翻史冊沐桀光。東亞第一偉太陽。疫蔓池臺無計奈，酒援經濟猛開張。母儀區夏於妺喜，興復中華比虞唐。莫受知識分子騙，救星誤給判豺狼。

    # 弔薩布林同志

    里加苦霧西溟渡，子夜舷窗一戰袍。鮑肆獨夫堪腐惡，遺風十月鬭天高。殺身因信須狂士，討篡防修要血潮。孤膽英雄求萬死，勞將大義諍愚曹。

    # ξекретарь頌集騶虞句

    勾踐一嚐清海寰。吹求搆陷任尖酸。細民猶樂十八稅，大桀方談百半年。須信治平仰聖意，謾因忠愛妬羣閹。祖龍若起水精裏，宜竄瘴荒留自殲。

    # 聞大陸行戲作

    都在甸服裏，恩讎未細聊。真龍儀袞冕，政客乞功勞。批準咨多士，回還教爾曹。選民需慎審，一票給天朝。

    # 品令·漢誓

    炮車相仕。須忠愛、隨生隨死。周全一帥何妨棄。孤將滅楚，役汝收失地。

    全體翻篇新主義。破江湖陳氣。勿謂言之而不預。內則擊斃。外有溝河峙。

    # 痛斥星鏈毒陰謀

    煞星稠掩母星空。犯境危國天不容。企是私營任捐棄，資非紅色理充公。本來共享蒼穹廣，何事專行霸道窮。定要盡誅蜥蜴黨，人民玉宇始澄清。

    # 驚聞喰種毒未清

    鬼子卡通黑闇殘。竟違文藝偉人言。宿讎亡我心非死，戰場闢之筆走間。文化彊國皆自信，廳局捷報喜相傳。一朝全滅流毒物，還看哪吒樣板片。

    # Not-stalgia

    少年對此以爲多。殘卷寥寥幾敝車。囹圄憶來還氣喘，豺狼強與事心魔。正招信手朝花去，恍若沈舟一世隔。流水衙前橋下月，至今猶恨不超脫。

    # 水調歌頭·仕辨

    機器真絕塔，醉紙勝勞牘。知余好隱，安須一再勸偪乎。不必援孰言志，各自平生稱意，知遠道行獨。況身經年業，逸興肯拋諸。

    溫牀鬧，鏽刃脆，敗肉烏。天德若此，不就洗煉罔相逐。坐對潮流袖手，致幻科層襁褓，難論現人族。秦例殘罡氣，未至比觀儒。

    # 斯大林頌

    斯大林傳真理音。有集體後有人民。繼遺志以列寧黨，敗匪軍於察里津。圍獵德國納粹獸，慢烹美帝北約禽。遠離控制心靈塔，對視之時死必臨。

    # 禁止躺平頌

    境外夷賊何日端。青年不振令心寒。勤勞本爲價實現，奮鬭談何升躍遷。洗腦挑唆須警惕，感恩奉獻自國安。齊開工享加班樂，彈指遠超美利堅。

    # 贈赴俄朝鮮人民軍諸達瓦里希

    抗美平烏事履冰。幸得義士洗戈兵。可憐遲暮故國後，不見英雄縱隊行。道路百年期再造，人民萬歲誓中興。階級兄弟齊殺狗，以爲全球報太平。

    # 五一偶感

    鳥獸池臺飾大荒。不由灌水取滄浪。數年熊市徵黃餅，十載蟻工點土鋼。後主蚤知閹宦惡，孤臣縱佔劍閣長。此中未悟逍遙理，枉唸南華頭一章。

    # 書憤

    馬謖空言餘漢淚，恪公自亢用譏嘲。須知辯證無窮法，萬古螺旋未可逃。

    羅素福柯利奧塔，間離羣衆太喧囂。平生不信馬跟列，墮入拔舌永不超。

    是與彼方倆是非。小兒作浪幾多回。自得賢者攀重否，天且非天萬象推。

    # 桂枝香·青年節

    心懷五四。最關鍵在乎，進步名義。打倒國賊亂政，靠諸學子。思潮反哺文人界，又聯合、工人階級。救亡之歲，危機動盪，一朝雄起。

    大街上、呼聲震地。是民衆潮湧，旌旗波舉。子彈兵車較此，但蚍蜉耳。土豪買辦都驚垮，俾民族、國恥新洗。便傳世界，青年集合，在長街裏。

    # 戲題

    廿八宿外滅明如。刻字石邊開闢初。民主儼然齊戶貫，富彊惜甚片膏腴。春深大院京和畿，日落帝國蜀與吳。願乞叮咚雞偉力，重開文化大天誅。

    # 軸心大捷贊

    蠻夷入則中華之。教士朝天猶羨奇。口岸紛爭既禮讓，弟兄團聚莫讎提。二分主畢相王事，東亞國宜自立極。命運之輪週轉下，共轄靈長贏分釐。
