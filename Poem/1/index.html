揠苗集乙巳稿

# 乙巳

# 滿江紅·誡己

一紙雄文，豈斗得、殊兵利器。長此理、非渠不懂，擲因偏執。驅以匕投煽事是，繼將賊勢危樓體。死方休、空洞罔思哲，昔盲矣。

融資畝，流金滴。天地轉，數值起。況僅此一家，先鋒國祇。波亂八紘偏晦養，目扶北斗直其指。待爭熄、術智各民得，當何日。

# 撞艦戲作

天兵大浪湧，犯境必擒拿。急遁小卒走，忽還三士家。魚鱉千里水，禮義艦頭華。硬氣王師共，衝量打鐵花。

# 原欲計學(Oeconomia Libidinis)三首

此派虛言奪霸權。寰球彊列豈消然。獨夫地堡銅京觀，僭主暴屍鬧市懸。庸衆沙文焉蹈火，鐵騎民粹尚蒙攔。必征餓虎無極鬭，草莽三餐諫勿專。

註：獨夫、僭主，希兒、墨子也。

此派虛言寧自繇。出逃橫縱恣風流。為公多士刃芒下，混戰大王奴隸頭。買辦怒燎換種志，擡尻樂作洋爹牛。苟成螳臂昨天事，菸復亭名且介抽。

此派虛言屠惡龍。未拔紅場克宮訇。難均頻犯寡人意，動亂卻勝私計雄。誰罔義豪窮五卷，膺存氣象舉八紘。不攖進運㬱激浪，更付榮懷一域中。

# 擬出遊歌

辭鄉之際，數日宴飲，至放曠得樂，引吭天外，散而還家，念各自東西，倏爾悲來，遂有所觸。

遊子別桑梓，意氣猶恍惚。湛湛秋雨冷，快意予形髑。少抿親朋酒，即日孑身途。出往異邦地，憾衆妄志諸。自負凌雲筆，欲奏伊呂疏。異邦多異人，風別性相殊。何以問嚶鳴，誰處求沫濡。乃得助興歌，饋我伯仲叔。

# 賀新郎·讀《論再生產》

類滅則國泯。起先鋒、領其風化，萬夫雄振。一念無非蒼生捄，昔者赤潮兆頃。貲本撫、多元綏靖。鐵幕徒泱師枉武，亦鏽蝕一觸訇然殞。赫日舛，庶黎馴。

事無兩樣人心遜。我同胞：進運是否，高而難問？毋懼槃結無端術，重把青年圍困。就頹爾、樓頭苦飲。萬事渺茫還真者，蓋有朝寰宇當明峻。解「是日，曷喪」恨。

# 題爲流溢編序

我未詢喚久，封侯凈無味。桌前百卷飲，陽臺菸拂月。閒點擸𢶍墨，妄志也得慰。

# 激寒

秋水激寒留所慍，日神含憊酒神驚。冰凌離亂蹀踱處，翠木飛禽尚舊聲。

# 水龍吟·基本盤

老鍾苦等龍吟，且聽且得有點久。大國寸止，亞空間裏，奸奇納垢。四五連拍，鍾生雄筆，以明黔首。要當歸當歸，小蛙蛙呀，快回來，陪茅酒。

須笑美國野狗。四分熟、不如粥豆。乞憐搖尾，騙局民主，實綜藝秀。還看大國，淨殺貪腐，爲民奔走。俺當征有召，星條旗下，多喰胡肉。

# 下場(Untergehen)引

君不見稼軒欄杆龍川樓，自古士人多斷腸。千頭紛擾安足慮，豎子舌條似婦長。善淫既是謠諑謂，何須自證投羅江。偈子非相諸，白骨累浮屠。下山蘇魯支，猶記基督多聖徒。左臉既蒙捶，賞君右面又何如。金絲菸捲十千酒，心事浩茫不可法狂輿。君不見先賢經綸今未傳，豈憑無言勝奴顏。

註：浮屠，佛陀、覺者。

# 爲普魯士統一德國吟弔黑格爾

居神勢運狂魔語，牧原靈氣曾默許。皇法尊者頌正書，草莽忽奏練兵曲。黑衫壓進玄鷹張，市民行會諗人王。諸夏同血早共振，目垂極野心飄颺。天兵不負馬背誓，焰火亂鞭破愚賊。漠野異族走亂轍，鐵紀男兒睨威色。連拔四郡潰奧法，統戰南部中歐合。容克升作彊權列，宰相肇造歸一國。王師飲馬萊茵水，席捲功堪大秦偉。二十五邦共我主，誰卻業竟罹憂退。詠懷太一流溢先，維人知性有發端。萬事迴歸絕對者，天父歆嗅此成全。

# 永遇樂·████大會上青年代表的講話

同志們哪，時局似袋，將露錐子。幽默蝗國，妖婆首相，頻爆咱雷子。青年吾輩，大敵臨近，不要只當擺子。請長纓，三軍令下，多殺日本鬼子。

手撕倭肉，京觀料理，開動大家筷子。可笑狂徒，妨華姦計，戲唱塌檯子。盡焚神社，過輪者死，除了櫻花妹子。倭平了，太平洋也，須加蓋子。

# 廿七日沙塵暴十一韻

北來踏野馬，江左竟邊音。穹壤朦朧色，煙霾熒惑暈。大食荒草漠，西部亂風村。埃土揚擊面，氣毒彌散魂。大顆肺放射，瘡敗點瘟神。福庇東南客，恩澤現代人。牲人來又往，勞動獻終身。切力早竭命，搬灰造眾倫。聖人不外戶，天下有黃塵。韃縱全千里，漢遺室內臣。以歌沙也厚，以頌沙之醇。

格律說明：排律。兩處異韻“聲”“城”，是方言故，已改；兩處三仄尾，是唐人律故。

# 題十二月七日《參考消息》

二分主、畢相王事；東亞國、宜自立極。

# 念奴嬌·苦海

鳳兮歸去，對案酌，兩筆萬端可解。蝸角鬭爭奚所欲，要劃員輿地界。人子相戕，老爺迭代，幾箇眞通略。至德何遠，竟長苦此天孽。

曰若不惡顓蒙，謂中有变力，超人應謔。理性猶然施狡計，無限憂懷怎寫。寰宇偃息，衆賢寥寂，熱血溫還卻。風流誰數，飲者已矣哉也。

# 滿江紅·賦乒乓贈Heliyoz

珠皓音鳴，謾誇可、撬開寰宇。記隻手、嘗征善戰，預謀盤計。擰擺衝劈都亂步，腦容費盡心餘力。要局中、遊走破囚籠，書靈氣。

連招數，纏鬭死。疊擺速，封牆峙。於技藝進取，更言癡似。重炮中臺重誘迫，閃失興歎弧圈跡。慨青春、恰此玉浮雷，狂飆去。
