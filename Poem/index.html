# 益卓毗阿集

# 歌詩

Ekropia: Lyrics

# 永遇樂·在████上的講話

同志們哪，今天我啊，開個會子。幽默島國，妖婆首相，耍嘴謅雷子。青年吾輩，大敵臨近，不要只當擺子。請長纓，金戈鐵馬，多殺日本鬼子。

手撕倭肉，京觀料理，開動大家筷子。█指揮槍，我軍字典，未有言敗字。盡焚神社，過輪者死，除了櫻花妹子。倭平了，太平洋也，須加蓋子。

# 爲普魯士統一德國吟弔黑格爾

居神勢運狂魔語，牧原靈氣曾默許。皇法尊者頌正書，草莽忽奏練兵曲。黑衫壓進玄鷹張，市民行會諗人王。諸夏同血早共振，目垂極野心飄颺。天兵不負馬背誓，焰火亂鞭破愚賊。漠野異族走亂轍，鐵紀男兒睨威色。 連拔四郡潰奧法，統戰南部中歐合。容克升作彊權列，宰相肇造歸一國。王師飲馬萊茵水，席捲功堪大秦偉。二十五邦共我主，誰卻業竟罹憂退。詠懷太一流溢先，維人知性有發端。萬事迴歸絕對者，天父歆嗅此成全。

# 下場(Untergehen)引

君不見稼軒欄杆龍川樓，自古士人多斷腸。千頭紛擾安足慮，豎子舌條似婦長。善淫既是謠諑謂，何須自證投羅江。 偈子非相諸，白骨累浮屠。下山蘇魯支，猶記基督多聖徒。左臉既蒙捶，賞君右面又何如。 金絲菸卷十千酒，心事浩茫不可法狂輿。君不見先賢經綸今未傳，豈憑無言勝奴顏。

註：浮屠，佛陀、覺者。

# 水龍吟·基本盤

老鍾苦等龍吟，且聽且得有點久。大國寸止，亞空間裏，奸奇納垢。四五連拍，鍾生雄筆，以明黔首。要當歸當歸，小蛙蛙呀，快回來，陪茅酒。

須笑美國野狗。四分熟、不如豇豆。乞憐搖尾，騙局民主，實綜藝秀。還看大國，淨殺貪腐，爲民奔走。俺當征有召，星條旗下，多喰胡肉。

# 激寒

秋水激寒有所慍，日神含憊酒神驚。冰凌離亂蹀踱處，翠木飛禽尚舊音。

# 我未詢喚久

我未詢喚久，封侯凈無味。桌前百卷飲，陽臺菸拂月。閒點擸𢶍墨，妄志也得慰。

# 賀新郎·讀《論再生產》

類滅則國泯。起先鋒、領其風化，萬夫雄振。一念無非蒼生捄，昔者赤潮兆頃。貲本撫、多元綏靖。鐵幕徒泱師枉武，亦鏽蝕一觸訇然殞。赫日舛，庶黎馴。

事無兩樣人心遜。我同胞：進運是否，高而難問？毋懼槃結無端術，重把青年圍困。就頹爾、樓頭苦飲。萬事渺茫還真者，蓋有朝寰宇當明峻。解「是日，曷喪」恨。

# 擬出遊歌

辭鄉之際，數日宴飲，至放曠得樂，引吭天外，散而還家，念各自東西，倏爾悲來，遂有所觸。

遊子別桑梓，意氣猶恍惚。湛湛秋雨冷，快意予形髑。少抿親朋酒，即日孑身途。出往異邦地，憾衆妄志諸。自負凌雲筆，欲奏伊呂疏。異邦多異人，風別性相殊。何以問嚶嚀？誰處求沫濡？乃得助興歌，饋我伯仲叔。

# 原欲計學(Oeconomia Libidinis)三首

此派虛言奪霸權。寰宇強列非消然。獨夫地壘銅京觀，僭主暴屍鬧市懸。「羣氓」何以沙文死？鐵騎尚蒙民粹攔。餓虎必征無盡鬭，毋專庶草三齋餐。

註：獨夫、僭主，二國君名。

此派虛言寧自繇。出逃橫縱恣風流。為公爾賢多遇刺，逍遙彼輩盡油頭。保貲怒發換種志，稽首樂爲美爹牛。苟成螳臂昨日事，且介亭邊復菸抽。

此派虛言屠惡龍。未摧紅場克宮訇。難均頻犯寡人意，動亂卻勝私計雄。誰罔義豪舉五卷，膺存氣象起八紘？進運不攖大浪㬱，更隨中域一懷榮。

# 撞艦戲作

天兵大浪湧，犯境必擒拿。急遁小卒走，忽還三士家。魚鱉萬里水，禮義艦頭華。硬氣同王師，鋼打衝量花。

# 滿江紅·誡己

一紙雄文，豈斗得、殊兵利器。長此理、非渠不懂，擲因偏執。驅以匕投煽事是，繼將賊勢危樓體。死方休、空洞罔思哲，昔盲矣。

融資畝，流金滴。天地轉，數值起。況僅此一家，先鋒國祇。波亂八紘偏晦養，目扶北斗直其指。待爭熄、術智各民得，當何日。

# 始文

始學文焉，但求通順，蓋文體相契謂之得也。仿古稽今，迥異徑庭諸言，莫有不略解者，蓋心有其像曰可，非欲熟焉。此所以明法度也。既專，則思言語無重，乃圖挪爾（novel）之科目諸元，進求饗效。飽讀語料，泰西之方家、中土之摩登，並蓄兼收，吾之志也。餘鄙力薄才，難爲工焉，請以散體敘之，蓋俱行文敷陳之事也。

又提筆時，向述斯事盡忘，全異於其讀演之默貌，然此果因之。惟心在於太虛，通達然彼相傳以詞字接踵。續詞生句，句連段成，後必返而度之，以目歷之，默然心誦，凡有不辭，盡刪易之，耳目俱相合乃止。凡句段之短長，皆有律當循，亦行刪易控之，以中規範。斯諸準則規範者，或謂脈也，或謂道也。蓋太虛有暴，有徵名雜，沿襲熟習謂之準也，能者因之乃有成。

末、竊姑妄試論筆法文意。子部之龍氣，太史之汪洋，後代多失；唐宋有復，然專於小物巧致，非於大宗。想嚴譯歐西諸子之時，伏案大爲犯難躊躇，其在此矣，亦彼可載科學之處也。今必喻文，則嗚呼諮嗟遍地，似牌匾焉，然其木早朽。其多未至初所謂通順也，更勿文筆是論。魯迅六篇、嚴譯八種，可謂近代雅言之至典也。有雲：今之大義，亦必以今之文詮。吾人其克從哉！

十月九日薄暮。

# 烏騅

余求學烏騅敗地，於一人一物，空空然無所感。畢目映而無違和，盡神遇而得其樂者，是謂適應，余也求之。夫其樂者，凡物各有其樂也，內在而自足。

［目映、神遇：海德格爾有apprehension“意領/體會”、comprehension“神會/理解”之語，此未之用。］

簡「建武三年、候粟君所責寇恩事」，皆案牘之言也，其意也機焉、其心也偶焉，散文方家，避之尤切。話必通於市井，故後世亦多流文辭之語，而鮮傳案牘之言。以是其今之視之，晦澀拗讀，因思或有死拉丁之名衆生綱目之效。

文自太虛來，太虛爲溝回共振者，謂得於妙手、謂「lisped in numbers, for the numbers came」。太虛者，有徵也，血肉槃結，筋脈扭錯，以是有言，因而生人，繼則成科學之人文。

［有徵，以言the Symbolic。］

「是故」猶今言「這導致」也；「必……則……」猶今言「一……就……」也。《古代漢語》雲，「猶言」猶今言「等於說」也。

夫「吾與子之所共適」雲者，非其風、非其月也。濠上也，有徵也，士間之以言相迴響也。

［士，以言subject，「臣」則過，「僕」則欠。］

九月二十一日午後風呂。

# 評江油事

其誠誡賊勢之危國體歟。

吾人其國是憂，恐內亂由外侵以為鍵也。無論事由，事有則必損，而爭論無益，蓋陷正義於口舌。馬哲云：「外因必以內因為介。」民憤所以除暴，除暴所以安良，安良所以禦敵，禦敵所以民富。故非民者，實保民也。故曰：誅五蠹於銃下者，挽諸民於大劫也。

然則必有先鋒，撫國維制，徐圖大同，此列寧主義之義也。若今先鋒是反，名喚伸張，然嘩鬧甚囂，究其根者，蓋久怨計生不景也爲固，有恐賊勢受用、篡逆遂行之實。先鋒反則進運背，有是實必連根拔。群氓庸眾，何信之有；非法非禮，不道即冠。嗚呼，歎劇變以降，甚矣普羅之教化也不逮！

贊曰︰

草謨檄兮膺無悔，逾邁催兮敵何止。

# 丑夜癲狂作

失道義者，江湖焉容！？

忍看朋輩兮，頓成新鬼；慨歎同門兮，一轉末人。

然「下場」一事，再未可犯。吾非蘇魯支也。然則去主體化者，去癔症化者也。墮視黜聽，方期不後歷史進運可矣。若夫譫妄剛愎，竟咄咄然以至於服羣氓者，自歸敗類也，救之安可？條理一道百，豕犬之輩，能奈若何！

悲也夫！

# 三界譯名記

拉康三界，悉所熟聞，然譯名不一。今鄙有策，雖一家妄言，或有可取。

夫最流行者，謂「實在界」、「想象界」、「符號界」又「象徵界」耳。「實在」、「想象」，延哲學中傳統之名，取而用則體系成，故異議極少；「象」重字於「想象界」，「徵」者同音而簡化之字也；而「符號」者「符目名號」者也，猶通古「名」，是以「象徵界」爲最下者，宜速摒棄。

然則原文，固無「界」言，所以名者、冠以形前而作抽象事也。拉康言「界」，獨稱「秩序」，所以定此三詞範疇者，鮮所比舉。古「有」作綴，冠專名也。古國名曰「有熊」、「有巢」；《詩》稱「有梅」、「有的」，俱出於此。因可作「有實」、「有像」、「有名」也。雖乏辨、歧義於耳口，文言行之洽矣。白固無冠，故幾能契，據舊白先例，或作「那實在底」、「那想象底」、「那符號底」，雖合法而盡意，粗俗視聽，其所不免。夫臺譯僅「實在」「想象」「象徵」耳，然形名互通，古來慣活；體用不常，今所正法。是頻發人誤解者，最不可用。

是以記之。

六月七日於浴室。
