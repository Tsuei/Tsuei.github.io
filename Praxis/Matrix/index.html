# 意识形态及哲学科学矩阵

Matrix of Ideologies, Philosophies and Wissenschaften

辩证唯物主义者学习哲学，最终是要导向意识形态批判或现代资本主义批判，也就是辩证唯物主义的理论工作。（注意理论与实践的关系 (opens new window)。）

作为唯物辩证法家，把握一定是体系化、环节化、阶次化的。

是以制表。

出于私心，我将人的思想真正能称为是哲学之前算作狭义的意识形态，广义的意识形态包含哲学；将辩证唯物主义认为科学。这就是标题的来源。
