nano是Linux的文本编辑器。

GCC（GNU Compiler Collection，GNU编译器套件）是由GNU开发的编程语言编译器。 只编译C程序，C++程序需要g++来编译。


编辑源代码：nano hellopp.cpp

编译C++至无后缀：g++ hellopp.cpp -o hellopp

打开文件的输出：./hellopp
